# MAST Taxonomy Reference

Classify each barrier's intervention type using these MAST chapters.
A single barrier can span multiple chapters — assign all that apply.

## Chapters

| Chapter | Domain | Key Types |
|---------|--------|-----------|
| **A** | SPS Measures | Sanitary/phytosanitary barriers |
| **B** | TBT | Technical barriers to trade |
| **D** | Contingent Trade-Protective | Anti-dumping (51), safeguards (52), countervailing duties (53) |
| **E** | Quotas, Prohibitions & Licensing | Import ban (22), licensing (36), quota (45) |
| **F** | Price-Control Measures | Government-mandated pricing, reference pricing, price caps |
| **G** | Finance Measures | Internal taxation of imports (48), including DSTs on foreign digital firms |
| **H** | Anti-Competitive Measures | Discriminatory enforcement, state monopolies |
| **I** | Trade-Related Investment | Local content (28), local operations/data localisation (29), trade balancing (34) |
| **L** | Subsidies & State Support | Production subsidy (5), tax relief (8), state aid (59) |
| **M** | Government Procurement | Preference margin (40), localisation (41), access restriction (42) |
| **N** | Intellectual Property | IP protection (24) |
| **P** | Export-Related Measures | Export tax (18), export ban (19), export licensing (35) |
| **FDI** | Foreign Direct Investment | Entry/ownership (25), treatment/operations (26), financial incentive (27) |
| **Tariff** | Import Tariff Measures | Import tariff (47), TRQ (44), internal taxation (48) |

## Confidence Levels

- **HIGH**: Clear, unambiguous match between the barrier and the MAST type
- **MEDIUM**: Plausible but indirect connection; the barrier could reasonably be classified here
